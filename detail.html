I WANT TO BUILD A WEBSITE:

Intro to Web Concepts:

  • Learn to use HTML and CSS - Read about the basic elements behind every website and how you can conceive and create your own website.
  • w3schools - Read and gain an overview of how the web works and what you will need to build to put your new website online.
  • General Assembly - Walk through basic design concepts and tasks in a highly interactive guided course.

    HTML:

  • Code Academy - Work your way through this highly interactive course that gives you a full understanding of HTML.
  • Web Design - Sign up for free daily or weekly lessons on understanding and using HTML in your web pages.
  • Know HTML - Read and review simple HTML information and examples to improve your HTML knowledge.
  • HTML with the Computer Guy - Eli the Computer Guy takes you through a video lecture introducing you to HTML.
  • w3schools - Read and interact with basic HTML language and concepts.

    • CSS:

  • w3schools - Review CSS concepts and examples, then create and test your own code before you build your site.
  • Code Academy - Make your way through these easily digestable courses and check your progress as you go.
  • Web design - Read your way through simple step-by-step instructions on creating your own CSS.
  • Know CSS - Read about what goes into CSS and check out simple but practical examples.
  • Computer Guys - Eli the Computer Guy introduces you to CSS in a short video you can view on your own time.

    • Javascript:

  • Code Academy - Start a beginner-level course in Javascript that you can complete in 10 hours - all at your own pace!
  • w3schools - Review a roadmap of how you can learn Javascript and tackle lessons in easy-to-manage steps.
  • Khan Academy - Follow your way through tested instructions videos that demonstrate basic Javascript concepts and knowledge.
  • Computer Guy - Eli the Computer Guy presents an introduction to web programming using Javascript.

    • Design Concepts

  • Design for Beginners
    • This page provides a list of articles to help you understand the basics. Read about the tools of the trade and explore a set of link to articles explaining the basics.
